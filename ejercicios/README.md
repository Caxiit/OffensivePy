# Ejercicios Python

Carpeta para practicar fundamentos de Python por niveles.

## Niveles sugeridos

```text
ejercicios/
├── basico/
├── intermedio/
└── avanzado/
```

## Básico

Temas recomendados:

- variables;
- tipos de datos;
- `if`, `elif`, `else`;
- bucles `for` y `while`;
- listas;
- funciones simples.

## Intermedio

Temas recomendados:

- diccionarios;
- tuplas y sets;
- manejo de archivos;
- manejo de errores con `try/except`;
- módulos;
- scripts de terminal.

## Avanzado

Temas recomendados:

- programación orientada a objetos;
- APIs;
- automatización;
- testing básico;
- proyectos completos.

## Convención de nombres

Usar nombres claros:

```text
01_variables.py
02_condicionales.py
03_bucles.py
04_funciones.py
```
